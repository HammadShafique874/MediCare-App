# Getting Started

Follow these steps to run the project on your local machine.

> **Prerequisites**
>
> Ensure you have completed the official React Native environment setup before running the project:
>
> https://reactnative.dev/docs/set-up-your-environment

## Step 1: Clone the Repository

```bash
git clone https://github.com/HammadShafique874/MediCare-App.git
cd MediCare-App
```

## Step 2: Install Dependencies

Using npm:

```bash
npm install
```

Or using Yarn:

```bash
yarn install
```

---

## Step 3: Start Metro Bundler

```bash
npm start
```

Or

```bash
yarn start
```

---

## Step 4: Run the Application

### Android

```bash
npm run android
```

Or

```bash
yarn android
```

### iOS

Install CocoaPods (first time only):

```bash
bundle install
bundle exec pod install
```

Run the app:

```bash
npm run ios
```

Or

```bash
yarn ios
```
